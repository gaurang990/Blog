# Blog
# 📝 Blog Web Application

A simple and responsive **Blog Web Application** that allows users to create, read, and manage blog posts. This project focuses on building a clean user interface and implementing core blogging functionalities.

## 🚀 Features

* 📰 Display blog posts
* ✍️ Create and manage blog content
* 📖 Read individual blog articles
* 🔍 Easy navigation between posts
* 📱 Responsive design
* 🎨 Clean and user-friendly interface
* ⚡ Fast and lightweight application

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

## ⚙️ Installation & Setup

### Clone the repository

```bash
git clone https://github.com/gaurang990/Blog.git
```

### Navigate to the project folder

```bash
cd Blog
```

### Run the project

Open the `index.html` file in your browser.

No additional installation is required.
